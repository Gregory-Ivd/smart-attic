# Smart Attic — лендинг копірайтера

Особистий односторінковий лендинг Григорія Йовдія (Smart Attic): копірайтинг,
лендинги під ключ, SEO, AI-процес. Трьохмовний — **UA / RU / EN**.

Статичний сайт, без бекенду. Хоститься на GitHub Pages (як testsAnna).

## Структура
- `index.html` — увесь лендинг + переклади + перемикач мов (один файл).
- `.nojekyll` — щоб GitHub Pages не чіпав файли через Jekyll.

## Як редагувати тексти
Усі тексти лежать у JS-словнику `T = { ua:{…}, ru:{…}, en:{…} }` всередині
`index.html` (внизу, у `<script>`). Кожен елемент на сторінці має атрибут
`data-i18n="ключ"` — при перемиканні мови підставляється `T[мова][ключ]`.

Щоб змінити текст — правиш значення під потрібним ключем у всіх трьох мовах.
Щоб додати новий блок — додаєш елемент з `data-i18n="новий_ключ"` і сам ключ у три мови.

Мова за замовчуванням — українська; вибір користувача запам'ятовується в `localStorage`.

## Деплой на GitHub Pages
1. Створити репозиторій на GitHub (напр. `smart-attic`).
2. `git remote add origin <url>` → `git push -u origin main`.
3. Settings → Pages → Source: `main` / root.
4. Сайт буде на `https://<user>.github.io/smart-attic/`.
